# @munchi/core

Core utilities and types for the Munchi SDK.

## Installation

```bash
npm install @munchi/core
# or
pnpm add @munchi/core
# or
yarn add @munchi/core
```

## Usage

```typescript
import { MUNCHI_CORE_VERSION } from '@munchi/core';

console.log(MUNCHI_CORE_VERSION);
```

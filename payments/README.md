# @munchi/payments

Payment processing utilities for the Munchi SDK.

## Installation

```bash
npm install @munchi/payments
# or
pnpm add @munchi/payments
# or
yarn add @munchi/payments
```

## Peer Dependencies

This package requires:
- `axios >= 1.0.0`

## Usage

```typescript
import { /* your exports */ } from '@munchi/payments';
```

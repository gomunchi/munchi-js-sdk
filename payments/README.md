# @munchi_oy/payments

Payment processing utilities for the Munchi SDK.

## Installation

```bash
npm install @munchi_oy/payments
# or
pnpm add @munchi_oy/payments
# or
yarn add @munchi_oy/payments
```

## Peer Dependencies

This package requires:
- `axios >= 1.0.0`

## Usage

```typescript
import { /* your exports */ } from '@munchi_oy/payments';
```
